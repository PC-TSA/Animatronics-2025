# Animatronics-2025
Complement to Ollivanders Ladder

## Serial Specs
Overall scheme: \<TYPE, NAME, VALUE\>

\<SERVO, SERVO\_NAME, SPEED, TIME (in ms)\>  
Servo name options:

- owl\_wing\_left  
- owl\_wing\_right

\<AXIS, AXIS\_NAME, box\>  
Axis name options, box options (left to right, top to bottom):

- Lr  
  - Left right axis  
- ladder  
  - Ladder axis

\<LIGHT, COLOR\>
